Services

Sites Web

Vous avez une idée géniale et voulez la partager ? Vous avez toujours voulu un site Web mais ne possédez pas les connaissances requises pour le faire. Vous avez une entreprise et voulez la mettre de l'avant ? Je ferai l'analyse de vos besoins et créerai le site Web parfait pour vous ou votre entreprise.

Hébergement Web

Vous avez un site Web, mais ne savez pas comment le rendre accessible au public ? Vous trouvez que vous payez beaucoup trop cher pour héberger votre site Web. ? Je vous offrirai un forfait d'hébergement adapté à vos besoins et à très bon prix. De plus, je m'occuperai personnellement de mettre votre site Web en ligne et m'assurerai de son bon fonctionnement.

Achat de nom de domaine

Vous avez un site Web, mais ne savez pas comment acheter un nom de domaine ? Vous ne voulez pas perdre votre temps à gérer votre nom de domaine ? Je trouverai avec vous le nom de domaine parfait pour mettre de l'avant votre site Web et je m'occuperai de tout ce qui est achat et gestion.

Applications mobiles

Vous avez déjà une application Web et voulez l'avoir sur mobile ? Vous voulez une application disponible sur Android et iOS ? À l'aide de Flutter ou de React Native, je peux rapidement développer une application mobile fonctionnelle autant sur Android que sur iOS.

Applications Web

Vous avez besoin d'un système de gestion pour votre entreprise ? Vous avez déjà un site Web et voulez avoir des fonctionnalités avancées comme la connexion ou l'inscription ? Je regarderai avec vous les fonctionnalités dont vous avez besoin et créerai l'application Web idéale pour vous.

Bases de données

Vous avez une base de données obsolète et voulez la mettre à jour ? Vous voulez migrer votre base de données vers un SGBD plus récent ou plus performant ? Je peux vous aider, car je maîtrise plusieurs bases de données, notamment : MongoDB, MySQL et SQLite.
